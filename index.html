고지서 → 엑셀 자동입력

평택지방해양수산청 납입고지서 PDF를 올리면 화물료·항만보안료·정산일을 읽어 업로드용 엑셀(Sheet3 서식)로 만들어 드려요.

01 선박명 ↔ 선박코드 매칭 테이블

▸

고지서의 선명(예: 나고아 트레이더)과 엑셀에서 쓰는 선박코드(예: NGTR)를 여기서 관리해요. 파싱한 선명이 목록에 없으면 표에서 바로 코드를 입력해 추가할 수 있어요.

02 MRN 대사표 업로드 (선택 — 항차 자동보정용)

항차가 자동으로 안 채워질 때를 대비해서, MRN 대사표(.xlsx)를 올려두시면 도움이 돼요.

📋
클릭해서 MRN 대사표(.xlsx) 선택 또는 여기로 드래그

03 고지서 PDF 업로드

한 파일에 여러 건(여러 페이지)이 섞여 있어도 괜찮아요. 이 PDF들은 텍스트가 아니라 그림(스캔/벡터) 형태라서 화면에서 직접 읽어내는 OCR 방식으로 처리해요 — 페이지당 몇 초씩 걸리고, 100% 정확하진 않으니 다운로드 전에 표에서 한 번씩 확인해주세요. 평택지방해양수산청의 표준 서식 기준으로 맞춰져 있어요. 항차가 고지서에 직접 인쇄되어 있지 않으면, 위에서 MRN 대사표를 올려둔 경우 자동으로 대사표 조회로 채우고 MRN 표시를 붙여드려요.

⚓
클릭해서 PDF 선택 또는 여기로 드래그
평택지방해양수산청 납입고지서(.pdf) · 여러 파일 가능 · OCR로 처리되어 다소 시간이 걸려요

04 파싱 결과 확인 & 수정

노란 배경 행은 확인이 필요해요. 값을 직접 고치거나, 선박코드가 비어있으면 입력 후 매칭 테이블에 저장할 수 있어요.
